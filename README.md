## FAQs

**Q1: How do I configure this bot for multiple eBay accounts?**  
You can import account credentials via the Appilot dashboard and manage each one under different device sessions.

**Q2: Does it detect negative reviews and respond differently?**  
Yes. The sentiment engine tailors responses for negative reviews, offering apology or resolution-based messages.

**Q3: Can I create my own response templates?**  
Absolutely. The system allows you to upload or modify response templates per account or sentiment type.

**Q4: Does it support scheduling replies at specific times?**  
Yes. The Smart Scheduler can automate reply posting windows for natural timing patterns.

**Q5: Is this automation safe for eBay accounts?**  
Yes, it mimics human-like actions and uses randomized delays, ensuring compliance with natural behavior patterns.
